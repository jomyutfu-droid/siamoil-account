สำรอง/กู้คืนข้อมูล

ข้อมูลทั้งหมดเก็บใน localStorage ของเบราว์เซอร์นี้เท่านั้น หากล้างประวัติ/แคชเบราว์เซอร์ ข้อมูลจะหายถาวร กรุณาสำรองข้อมูล (ดาวน์โหลดไฟล์ .json) เป็นประจำทุกสัปดาห์ และเก็บไฟล์ไว้ในที่ปลอดภัย

ซิงก์ข้อมูลผ่าน GitHub

ใช้ GitHub private repo เป็นที่เก็บข้อมูลกลาง เพื่อให้เปิดโปรแกรมจากคอมพิวเตอร์และมือถือเห็นข้อมูลเดียวกัน ต้องสร้าง Personal Access Token ด้วยตัวเอง (ดูวิธีด้านล่าง) — โปรแกรมนี้จะไม่สร้างหรือขอโทเค็นให้เด็ดขาด

ยังไม่เคยซิงก์
วิธีสร้าง Personal Access Token (ทำครั้งเดียวต่อเครื่อง ห้ามส่งให้ใครรวมถึง AI):
1. เข้า GitHub → รูปโปรไฟล์มุมขวาบน → Settings
2. เมนูซ้ายล่างสุด → Developer settings
3. Personal access tokens → Fine-grained tokens → Generate new token
4. Repository access: เลือก Only select repositories → เลือกเฉพาะ siamoil-account-data
5. Permissions → Repository permissions → Contents = Read and write
6. อายุ token ตั้งได้สูงสุด 1 ปี แล้วกด Generate token คัดลอกมาวางในช่อง Token ด้านบน
7. ห้ามวาง token นี้ในที่อื่นหรือแชร์ให้ใคร (รวมถึง AI/แชทใดๆ) — เก็บเป็นความลับเหมือนรหัสผ่าน

ขอบเขตของโปรแกรม v1

นอกขอบเขต v1: เชื่อมธนาคารอัตโนมัติ, e-Tax Invoice, หลายผู้ใช้/หลายเครื่อง, สต๊อกสินค้า, เงินเดือน, งบดุล/บัญชีแยกประเภทเต็มรูป

คำเตือนสำคัญ: โปรแกรมนี้ช่วยงานเอกสาร+รายงานภายใน แต่การปิดงบส่ง DBD/สรรพากรยังต้องผ่านสำนักงานบัญชี และควรใช้คู่กับ FlowAccount (จ่ายแล้วถึง มี.ค. 2027) อย่างน้อย 1–2 เดือนเพื่อเทียบผลก่อนเลิกใช้จริง
